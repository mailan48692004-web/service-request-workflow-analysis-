# Service Request Workflow Performance Analysis
Operational analytics project using service request data to identify delays, workflow bottlenecks and opportunities to improve service delivery.
## Overview
This project analyses service request workflow data to understand where delays occur, identify operational inefficiencies, and suggest improvements to service delivery performance.
## Business Problem
Service request systems often face delays, backlogs, and inconsistent completion patterns. This project examines service request data to identify bottlenecks in the workflow and provide data-driven recommendations for improving monitoring and response efficiency.
## Dataset
The dataset contains service request records, including variables such as request category, date received, date completed, suburb, and days to complete.
